核心功能

提供完整的前端開發工具鏈和最佳實踐

Vite 建置工具

使用 Vite 提供極速的開發體驗，支援熱重載、快速建置和最佳化輸出。

  • 極速的熱重載
  • 生產版本最佳化
  • 多頁應用支援
  • TypeScript 支援

Bootstrap 整合

基於 Bootstrap 5.3 的響應式設計系統，提供豐富的組件和工具類別。

  • 響應式網格系統
  • 豐富的 UI 組件
  • 工具類別系統
  • 自訂變數支援

SCSS 架構

採用 BEM 命名規範的 SCSS 架構，確保樣式的可維護性和擴展性。

  • BEM 命名規範
  • 模組化樣式
  • 變數系統
  • Mixins 工具

無障礙設計

  • 適當的對比度

開始使用

只需三個簡單步驟，立即開始您的前端開發之旅

1

下載專案

從 GitHub 克隆或下載 Case Template 專案到您的開發環境

2

安裝依賴

執行 npm install 安裝所有必要的開發依賴套件

3

開始開發

執行 npm run dev 啟動開發伺服器，開始您的開發工作
